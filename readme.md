A simple CSS helper file with some classes for margin, padding and font size.

You can use this CDN links to get the css file.
	<a href="https://cdn.jsdelivr.net/gh/Mainul12501/css-common-helper-classes/helper.css">JSDelivr</a>

CSS class name structures:

* FONT SIZES 
	.f-s-x (x = 1 to 150)

* MARGIN TOP
	.m-t-x (x = 1 to 300)
	
* MARGIN BOTTOM
	.m-b-x (x = 1 to 300)
	
* MARGIN RIGHT
	.m-r-x (x = 1 to 300)
	
* MARGIN LEFT
	.m-l-x (x = 1 to 300)
	
* PADDING TOP
	.p-t-x (x = 1 to 300)
		
* PADDING BOTTOM
	.p-b-x (x = 1 to 300)
		
* PADDING RIGHT
	.p-r-x (x = 1 to 300)
		
* PADDING LEFT
	.p-l-x (x = 1 to 300)
